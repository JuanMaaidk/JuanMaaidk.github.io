HTML

HTML (Lenguaje de Marcado de Hipertexto) es el lenguaje estándar utilizado para crear y estructurar el contenido de las páginas web en Internet. Proporciona un marco para organizar y mostrar texto, imágenes, enlaces, multimedia y otros elementos en un navegador web.

CSS

CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la apariencia y el diseño de un documento HTML o XML. Mientras que HTML define la estructura y el contenido de la página, CSS se encarga de la parte visual, como colores, fuentes, espaciado y distribución de elementos.
